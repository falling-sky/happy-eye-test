Browse the user agent stats, filtered by regex and minimum count. Regular expressions are javascript based. You can adjust the "minimum", which will require that a given user agent appeared several times in my logs before being in this list. 25 appears to be a good baseline number that filers out many preproduction browser builds.

Data: logs from test-ipv6.com where user was positively dual stack from the last 90 days. Disclaimers: Users visiting site are self-selecting and biased to check their IPv6. Overall numbers are small to the point of probably not being statisticaly useful. But sometimes this stuff is fun.
